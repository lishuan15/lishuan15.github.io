- # 近停视界开放API

  ## 欢迎进入近视防控的智慧未来

  欢迎访问**近停视界开放API**文档中心。在循证医学证据井喷、创新技术层出不穷的今天，近视防控正在经历一场划时代的变革。作为这场变革的**开创者、共建者和见证者**，近停视界致力于构建一个“眼科医生集团 + 4L甄选平台”双核驱动的智慧健康防控体系。

  现在，我们将这一体系的核心能力通过API对外开放。我们诚挚邀请医疗机构、视光中心、硬件厂商及健康服务开发者接入我们，共同将“让中国好孩子，都有一双好眼睛”的愿景，写进中国近视防控史。

  ## 平台简介

  近停视界不仅仅是一个服务平台，更是一套完整的智慧健康近视防控生态体系。通过开放API，您可以获得以下核心能力：

  - **专业赋能**：整合国内丰富4L防控经验的眼科医生资源，将专业的远程诊断评估、个性化方案制定及实时答疑的能力，封装成标准化的服务接口。
  - **产品严选**：接入“4L甄选平台”的数据能力，获取从功效、安全、临床验证、依从性等多维度严选的防控产品信息，为您的用户提供可靠的选择。
  - **全周期管理**：复现“赵阳眼科”级的全周期健康管理服务流程，通过API驱动医生、视光师、客服的全程陪伴，实现抗近有保障，家长更放心。
  - **生态构建**：助力合作伙伴打造**科普系统化、筛查精细化、干预个性化、技术循证化、服务全域化、效果可视化**的防控生态体系。

  ## 快速接入

  ### 1. 请求结构

  所有API请求均通过HTTPS协议进行，确保数据传输的安全性。请将请求发送至以下基础URL：

  ```
  https://store.myopia88.com/dev-api/***
  ```

  ### 2. 身份认证与鉴权

  近停视界开放API使用 **API Key** 进行身份认证。

  - **获取方式**：请联系我们的技术团队申请您的专属`RSA公钥`和`APIkey`。
  - **使用方式**：在HTTP Header中携带您的认证信息。

  ### 3. 调用频率限制

  为保证服务的稳定性和公平性，我们对API的调用频率进行了限制。默认情况下：

  - **基础频率**：每个`APIkey` 4次/秒。

  ### 4. 响应格式与状态码

  所有API响应均以JSON格式返回。

  - **成功响应 (200)**：

    json

    ```
    {
      "code": 200,
      "msg": "操作成功",
      "data": { ... }
    }
    ```

    

  - **错误响应 (4xx/5xx)**：

    json

    ```
    {
      "code": 500,
      "msg": "具体的错误描述",
      "data": "null"
    }
    ```

    

  常见HTTP状态码：

  | 状态码 | 含义                  | 处理建议                      |
  | :----- | :-------------------- | :---------------------------- |
  | 200    | OK                    | 请求成功。                    |
  | 401    | Unauthorized          | 认证失败，API Key无效或缺失。 |
  | 403    | Forbidden             | 无权限访问该资源。            |
  | 500    | Internal Server Error | 具体的错误描述                |
