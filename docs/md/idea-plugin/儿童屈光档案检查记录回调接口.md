# 儿童屈光档案检查记录回调接口

## 1. 接口概述

当新增儿童屈光档案检查记录成功后，系统会异步回调第三方系统，通知检查记录信息。

### 1.1 基本信息

| 项目             | 说明                         |
| :--------------- | :--------------------------- |
| **接口名称**     | 儿童屈光档案检查记录回调接口 |
| **请求方式**     | POST                         |
| **请求地址**     | `由调用方提供`               |
| **Content-Type** | `application/json`           |

### 1.2 接口特征

- **调用时机**：新增儿童屈光档案检查记录成功后
- **调用方式**：异步调用，不影响主业务流程
- **数据格式**：请求为JSON格式
- **字符编码**：UTF-8

------

## 2. 请求说明

### 2.1 请求头（Headers）

| 参数名       | 必填 | 类型   | 说明                     |
| :----------- | :--- | :----- | :----------------------- |
| Content-Type | 是   | String | 固定值：application/json |

### 2.2 请求体（Body）

| 字段名         | 类型   | 必填 | 说明                       | 示例值             |
| :------------- | :----- | :--- | :------------------------- | :----------------- |
| optometryId    | Long   | 是   | 屈光档案检查记录ID         | 123456             |
| age            | String | 是   | 检查时的年龄（几岁几个月） | 5岁3个月           |
| height         | String | 否   | 身高（cm）                 | 110                |
| weight         | String | 否   | 体重（kg）                 | 50                 |
| examineTime    | String | 是   | 视光检查时间               | 2024-01-15         |
| examineAddress | String | 否   | 检查地址                   | 北京市朝阳区XX医院 |
| eyeAxisOd      | String | 否   | 右眼眼轴                   | 22.5               |
| eyeAxisOs      | String | 否   | 左眼眼轴                   | 22.3               |
| odK1           | String | 否   | 右眼K1值                   | 42.5               |
| odK2           | String | 否   | 右眼K2值                   | 43.2               |
| osK1           | String | 否   | 左眼K1值                   | 42.8               |
| osK2           | String | 否   | 左眼K2值                   | 43.5               |

### 2.3 请求示例

json

```
{
  "optometryId": 123456,
  "age": "5岁3个月",
  "height": 110,
  "weight": 50,
  "examineTime": "2024-01-15",
  "examineAddress": "北京市朝阳区XX医院",
  "eyeAxisOd": "22.5",
  "eyeAxisOs": "22.3",
  "odK1": "42.5",
  "odK2": "43.2",
  "osK1": "42.8",
  "osK2": "43.5"
}
```



------

## 3. 响应说明

### 3.1 响应参数

| 字段名  | 类型    | 说明                            |
| :------ | :------ | :------------------------------ |
| code    | Integer | 状态码（200：成功，其他：失败） |
| message | String  | 响应消息                        |

### 3.2 响应示例

#### 成功响应

json

```
{
  "code": 200,
  "message": "success"
}
```

------

## 4. 调用方需提供信息

调用方需要提供以下信息以便配置回调地址：

| 参数    | 说明                   | 示例                                        |
| :------ | :--------------------- | :------------------------------------------ |
| 回调URL | 接收回调数据的接口地址 | `http://xxx.xxx.com/api/callback/optometry` |

## 5. 回调机制说明

### 5.1 调用时机

- 新增儿童屈光档案检查记录成功后

### 5.2 重试机制

- 回调失败不会重试（避免影响主业务流程）
- 建议调用方自行处理数据同步或后续查询

## 6. 错误码说明（调用方需返回）

| 状态码 | 说明     | 处理建议                   |
| :----- | :------- | :------------------------- |
| 200    | 接收成功 | 系统记录成功日志           |
| 其他   | 接收失败 | 系统记录失败日志，但不重试 |
