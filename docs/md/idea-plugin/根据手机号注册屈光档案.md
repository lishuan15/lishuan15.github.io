# 儿童档案注册接口

## 1. 接口概述

根据手机号注册账号并新增儿童用户档案。

### 1.1 基本信息

| 项目             | 说明                                                       |
| :--------------- | :--------------------------------------------------------- |
| **接口名称**     | 儿童档案注册接口                                           |
| **请求方式**     | POST                                                       |
| **请求地址**     | `http://127.0.0.1:8080/system/openapi/childArchives/mpAdd` |
| **Content-Type** | `application/json`                                         |

### 1.2 接口特征

- **鉴权方式**：请求头携带RSA加密后的API-Key
- **数据格式**：请求/响应均为JSON格式
- **字符编码**：UTF-8

------

## 2. 请求说明

### 2.1 请求头（Headers）

| 参数名                 | 必填 | 类型   | 说明                     |
| :--------------------- | :--- | :----- | :----------------------- |
| JTSJ-API-Key-Encrypted | 是   | String | RSA加密后的API-Key密文   |
| Content-Type           | 是   | String | 固定值：application/json |

### 2.2 请求体（Body）

| 字段名       | 类型   | 必填 | 说明                                                         | 示例值        |
| :----------- | :----- | :--- | :----------------------------------------------------------- | :------------ |
| nickName     | String | 是   | 儿童昵称                                                     | 张三          |
| birthdayTime | String | 是   | 出生时间戳（毫秒）                                           | 1477670400000 |
| sex          | String | 是   | 性别（0：男 1：女）                                          | 0             |
| highMyopia   | String | 是   | 是否高度近视（0：均非高度近视；1：一方高度近视；3：双方高度近视） | 0             |
| phonenumber  | String | 是   | 手机号                                                       | 13121528888   |

### 2.3 请求示例

json

```
{
  "nickName": "张三",
  "birthdayTime": "1477670400000",
  "sex": "0",
  "highMyopia": "0",
  "phonenumber": "13121528888"
}
```



------

## 3. 响应说明

### 3.1 响应参数

| 字段名 | 类型    | 说明                |
| :----- | :------ | :------------------ |
| code   | Integer | 状态码（200：成功） |
| msg    | String  | 响应消息            |
| data   | Object  | 返回数据（可为空）  |

### 3.2 响应示例

#### 成功响应

json

```
{
  "code": 200,
  "msg": "操作成功",
  "data": "2029392334912679938" //儿童档案ID，对应回调接口中的childArchivesId
}
```



#### 失败响应示例（参数错误）

json

```
{
  "code": 500,
  "msg": "手机号不能为空",
  "data": null
}
```



#### 失败响应示例（手机号格式错误）

json

```
{
  "code": 500,
  "msg": "手机号格式不正确",
  "data": null
}
```
